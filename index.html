About Me

For as long as I remember, I have been actively learning how to code different languages ever since I fell in love with coding when I was first introduced in my freshman year of high school. I was learning scratch programming with the introduction of if statements and while loops, being a math fanatic, I remember loving to trace through each iteration of the loop and the satisfaction that arose when I finished calculating a value and ran the program just to see that the value I calculated was indeed correct. Projects

Search Engine

With the help of a fellow developer, we developed our own custom search engine which took websites and parsed their link, title, and description, into 3 main file formats of your choice: CSV, JSON, and XML. The search engine was also able to revert those formats back into a readable list to be outputted back to the user interface.

Game Hub (Pong)

As a school project, my group decided to make a game hub in which we allowed users to play some of their favorite arcade games like pong. We made it so that the user's score can be put into a database and they can compete with other users. To achieve this, we made a login authentication with user tokens and only once the user is logged in, are they able to update their score to the database.

React Clicking Game

Made a clicking game with react where it contained 12 images and the objective of the game was to test the user's memory and click cards that the user never clicked before.
